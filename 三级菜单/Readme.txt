#项目背景/作用
三级菜单基于老男孩培训作业开发；
1.进入时打印一级菜单；
2.根据 id 输入打印对应的二级和三级菜单，输入 b 返回上级菜单，输入 q 退出程序。

#更新日志：
首次发布，无更新；

#原理说明：
1.数据存储在字典内；
2.使用 sorted 将字典内的元素排序给列表；
3.输入 id 后，根据 id 打印对应的 value 值；
4、输入 b 退出 本次for 循环，输入 q 退出程序。

#使用方法：
1.运行程序，打印一级菜单；
2.输入 id， 打印对应的二级菜单；
3.输入 id，打印对应的三级菜单；
4.输入 id，打印对应的 value值；
5.在除一级菜单之外任意菜单，输入 b 返回上级菜单，输入 q 退出程序。
