#项目背景/作用
用户输入程序基于老男孩培训作业开发；
1.用户正常输入可以登录；
2.连续输入三次锁定账户。

#更新日志：
首次发布，无更新；

#原理说明：
1.用户名和密码都保存在文件内，用户输入用户名和密码时，首先判断用户名密码是否存在于黑名单内，如果存在，弹出锁定提示，结束程序；
2.如果不存在于黑名单内，下一步读取用户名密码文件，进行对比，正确时弹出登陆成功提示，结束程序；
3.如密码错误，继续允许尝试，超过三次，弹出提示，加入黑名单，并结束程序。

#使用方法：
1.运行程序，输入正确的用户名密码，内置用户名为 zhangsan，密码为123456，登陆成功；
2.运行程序，输入三次错误的密码，账户被锁定；
3.运行程序，输入正确的密码或者错误的密码，账户被锁定。
